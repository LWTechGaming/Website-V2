Projects

I participate in many projects, most of them programming related. Some are part of the Free Open Source Software realm, while some are internal only. Most are voluntary, few are paid. On this page, projects where I participate(d) and maintain(ed) are displayed.

Positions

Oy Tarra-Tac Ab

Software developer, systems administator (Part-time)
Tarra-Tac is my father's print shop where I work on and off as a systems administator. I also develop some software and dedicated automation solutions for the company. This is where I have primarily gained my "real" work experience, if you don't count volunteer projects.

TheSharks

Lead documentation writer, developer, translator
TheSharks is an online guild of developers that has created the chat bot WildBot for Discord. At TheSharks, I'm in charge of the documentation and translations. I also do development and remote systems administation for our server infrastructure. TheSharks was my first OSS participation.

DevLodge

Lead developer, organization head, administator
DevLodge is a duo consisting of myself and my online friend Curtis from the US. Before uniting to work on duo projects, we were both developers at TheSharks. While most of our projects are handled by Curtis nowadays, I participated actively there before chasing other prospects.

Projects

website-boilerplate

JavaScript
A simple, lightweight website development environment with live reloading, built around live-server.

printcalc-api

JavaScript, Express.js
An API to host data related to PrintCalc, a program I'm developing at work. Not much here at the time of writing, more coming soon.

data-safe

JavaScript, ShareX
A non-weeb fork of lolisafe by the WeebDev team. Powers my personal image server, uses the slightly aged 2.x version of lolisafe.

Discord-Spotify

JavaScript, discordie
A small CLI script that runs on your Discord user account. Displays the current playing track in the playing status. A native integration for doing this is coming soon.

Swordfish & Logger

JavaScript, discordie/Eris
Two Discord bots created within DevLodge, these bots are nowadays maintained by Curtis F. and are closed source. I lead the development of the first versions.

postgresql-portable-lite

JavaScript
This is a (Truly) portable version of PostgreSQL-Portable by garethflowers. It doesn't need extra software to work and downloads additional components automatically.

Contributions

JavaScript, discord.js
This Discord bot powered Muselk's Discord server for a long time. After some internal controversy, it was deprecated and is no longer maintained.

electron-release-server

JavaScript, Electron, Sails.js
A Node.js web service that serves & manages releases of an Electron app, fully compatible with the Squirrel auto-updater built into Electron.

Grimoire & GrimoireWeb

Java, JavaScript, React.js
The Discord bot Grimoire and the website associated with it belong to my online friend BeMacized, with whom I used to develop the previously mentioned Midnight.
